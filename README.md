# Github Jobs App

基于 github-jobs-api 的 React 应用

## 项目地址

https://github-jobs-app.vercel.app

## 技术栈

- React，使用 React Hooks 编写组件逻辑和自定义 Hooks([useFetchJobs](https://github.com/wjx0820/Github-Jobs-App/blob/master/src/useFetchJobs.js))
- React Bootstrap 组件库和 React Icons 图标库，完成整个应用的布局，编写按钮，卡片，搜索表单，分页组件等
- axios，发送接口请求，以及[撤销冗余请求](https://github.com/axios/axios#cancellation)

## API 接口

[GithubJobsApi](https://jobs.github.com/api)

## 项目启动

```
npm install
npm run start
```
